## ðŸš€ Getting Started (Run your own *Mandali*)

Want to run this *tamasha* on your local machine?

1.  **Clone the repo:**

    ```bash
    git clone https://github.com/your-username/maskbook.git
    cd maskbook
    ```

2.  **Start the Backend (Go):**

    ```bash
    cd server
    ```

      * Create a `.env` file and add your variables:
        ```env
        PORT = "8080"
        MONGO_URI = "your_mongodb_connection_string"
        JWT_SECRET = "a_secret_safer_than_your_family_recipes"
        ```
      * Install dependencies and run:
        ```bash
        go mod tidy  # Fetches dependencies
        go run main.go
        ```
      * Your Go backend is now running on `http://localhost:8080`

3.  **Start the Frontend (Svelte):**

    ```bash
    cd ../frontend  # Go back to root and into client
    npm install
    npm run dev
    ```

      * Your Svelte app is now running, probably on `http://localhost:5173`.

4.  Open your browser to the Svelte app's URL and start the *bakar*\!
